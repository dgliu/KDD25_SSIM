## Environment Requirements

- python: 3.8
- tensorflow: 2.10
- numpy: 1.24
- pandas: 1.5
- scikit-learn: 1.2
- scipy: 1.10


## Run
Running HMoE Model:
```
# For AliCCP
python trainer.py --dataset ali-ccp
```

Running SSIM in the HMoE Model:
```
# For AliCCP
python SSIM3trainer.py --dataset ali-ccp
```
